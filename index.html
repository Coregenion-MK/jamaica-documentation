Jamaica

Jamaica-flutter

A new Flutter project for the Jamaica Flutter App (Android + IOS).

Prerequisites

Before running the app, make sure you have the following installed:

* Flutter SDK: Follow the official [Flutter installation guide](https://flutter.dev/docs/get-started/install) for your operating system.
* This application currently runs on Flutter version 3.10.5.

Getting Started

To get started with the app, follow these steps:

1. Clone the Repository

Clone this repository by running the following command in your terminal:

`git clone https://github.com/info-coregenion/jamaica-flutter.git jamaica-flutter`

2. Navigate to the Project Directory

Navigate to the project directory using the `cd` command:
It would be like 'cd jamaica-flutter'

3. Fetch Dependencies

Fetch the required dependencies for the project by running the following command:

'flutter pub get'

4. Configure Flavours

This application has three flavours. Configure the flavours in the edit configuration.

1- Got to the main button dropdown located at the left of run button
2- Select edit configuration and write the flavor name there and save.
3- To avoid any issue go with 'PROD'

### The flavours are

- QA
- PROD
- DEV
Sound Null safety Command should be added. As currently there are some packages that are not fully support null safety.

6. Run the App

Run the app on a connected device or emulator by executing the following command:

Execute `flutter run` in the project directory.

Make sure you have a connected device or an emulator set up before running the command.

Setting up emulator

1. Open AVD Manager: Click on the AVD Manager icon in the toolbar, or go to "Tools" > "AVD Manager."

2. Create a Virtual Device: In the AVD Manager window, click on the "Create Virtual Device" button.

3. Select Device Type: Choose a device type from the list (e.g., Pixel, Nexus). Click "Next."

4. Choose a System Image: Select the system image you want to use for the emulator. 
   It's recommended to choose one with Google Play Services if you plan to use Google APIs. Click "Next."

5. Configure Virtual Device: Enter a name for the virtual device and adjust other settings like RAM,
   storage, and orientation. Click "Finish" when you're done.

6. Launch the Emulator: In the AVD Manager, click on the play button (green triangle) to start the emulator.

7. Run Your Flutter App: In Android Studio, select your Flutter project from the dropdown menu in the
   toolbar. Click on the run button (green triangle) or use the keyboard shortcut (Shift + F10) to run
   your app on the emulator.

The emulator will take some time to start up, and once it's ready, your Flutter app should launch on the
emulator.

Note: If you encounter any issues or errors during the setup process, make sure to check the Flutter and
Android Studio documentation for troubleshooting steps.
